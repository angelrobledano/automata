# 1. Clonar el Proyecto

Bienvenido al equipo. Sigue estos pasos para obtener el código base en tu máquina.

## Requisitos previos
- Git instalado.
- Acceso a GitHub configurado con clave SSH.
- Node.js v18+.

## Instrucciones
```bash
git clone git@github.com:empresa/agente-pedidos.git
cd agente-pedidos
npm install
cd dashboard && npm install && cd ..
```
