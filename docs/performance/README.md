# Performance y SLAs

- **Webhook Response Time**: < 50ms (Mandatorio para evitar baneos de Meta).
- **RAG Retrieval**: < 200ms (P95).
- **OpenAI Generation**: ~2.5 segundos a 5 segundos.
- **End-to-End Latency**: Un usuario de WhatsApp debe recibir su respuesta en menos de 6 segundos desde que aparece el doble check.
