# Glosario Ubicuo

- **Tenant**: El cliente B2B (Commerce).
- **WABA**: WhatsApp Business Account.
- **phone_number_id**: El identificador interno de Meta para el número de WhatsApp.
- **Embedding**: Vector numérico de 1536 dimensiones que representa el significado semántico de un texto.
- **Chunk**: Fragmento de texto de ~500 caracteres, extraído de un documento base.
- **Semantic Cache**: Base de datos de preguntas frecuentes ya respondidas por IA para ahorrar tokens.
- **Handoff**: Acción mediante la cual un humano toma control del chat, desactivando la IA.
